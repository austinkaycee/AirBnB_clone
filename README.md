## Overview
This project is a simplified Airbnb clone backend, developed as part of the ALX Software Engineering program. It implements a command-line interface to manage users, places, amenities, and reviews using object-oriented design and persistent storage.

**Tech stack:** Python, OOP, JSON-based storage  
**Key features:** Custom command interpreter, data models, serialization/deserialization, unit testing
